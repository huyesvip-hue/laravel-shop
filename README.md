# Laravel Shop Giày

## Giới thiệu
Dự án website bán giày được xây dựng bằng Laravel.

## Công nghệ sử dụng
- Laravel
- MySQL
- JavaScript
- CSS / Bootstrap / Blade
- Vite

## Chức năng chính
- Đăng ký / đăng nhập
- Quản lý sản phẩm (CRUD)
- Giỏ hàng
- Đặt hàng
- Quản lý đơn hàng
- Phân quyền Admin / User
